Дерево описывается в формате JSON.
Каждый узел дерева - это поле БД, которое можно создать и от которого можно перейти к другим соответствующим полям.

Возьмем следующий пример:

{
  "cinema": {
    "name": "varchar" ,
    "address": "varchar" ,
    "hall": {
      "name": "varchar",
      "seats": "int",
      "session": {
        "time": "datetime"
      }
    }
  },
  "film": {
    "name": "varchar"
  }
}

cinema и film идут от корня дерева. В них описываются поля данных таблиц и также внутри cinema есть hall. Другая таблица, которая в свою очередь хранит свои поля и которая является child для cinema.
